DreamBooth

  • Main Method (Imagen)：

    • Fine-tune whole T2I model & Optimize the rare token embedding

    • Class-specific Prior Preservation Loss, combines samples generated by the T2I model, which are in the same category as the new concept, with given training samples as training dataset

  • Main Method (Stable Diffusion):

    • Fine-tune K\V projection matrices for all cross-attention layers & Optimize token embedding

    • Using CLIP-Retrieve to construct Regularization dataset

    • Joint Training or Close-Formed Merge of Separate Training for Multiple-Concept Compositiona

Mix of Show

  • Observation:

    • The embeddings are capable of capturing concepts within the domain of pretrained models, while the LoRA helps capture out-domain information. (Compare a,b & c)

    • Existing LoRA weights encode most of the concept identity and project semantically similar embeddings to visually distinct concepts, leading to conflicts during concept fusion. (Only c)

    • Weight fusion leads to identity loss of individual concepts in concept fusion. (Compare e & f)

  • Main Method (Stable Diffusion + LoRA):

    • ED-LoRA To preserve more in-domain essence within the embedding while capturing the remaining details using LoRA weights, adopt a layer-wise embedding and create a multi-word representation for the concept token V=Vrand+Vclass+

    • Gradient Fusion Preserve the identity of each concept in the fused model by aligning their single-concept inference behavior, W=arg⁡minW∑i‖(W0+ΔWi)Xi−WXi‖Fσ2

    • An excellent example of writing in the “Observation-Reason-Solution” format.

Cones → All that glitters is not gold

  • Main Method (Custom Diffusion):

    • If reducing the activation level of a specific neuron results in a decrease in the DreamBooth Loss associated with a particular new concept, such a neuron is designated as the concept neuron corresponding to that new concept.

    • For each new concept, within the Key/Value Projection Matrices of each Cross-Attention layer in the SD, the set of concept neurons associated with the new concept is identified. By setting the activation level of these neurons to zero, thereby effectively masking them, the new concept can be integrated into the SD model.

    • For a group of new concepts, the identification of the concept neuron sets corresponding to each new concept and the subsequent union of these sets constitute the collective concept neuron set for that group of new concepts.

  • Deep Insight:

    • Is the concept neurons of "Realtion" and ones of "Subject" perfectedly disentangled ?

    • Design of an Adaptive Concept Neuron Localization Algorithm

Object Encoder

  • Main Method (Imagen):

    • Training Stage Given a domain-specific image dataset, introduce a Pre-trained image captioning model to generate text corresponding to each image. Deploy a trainable Object Encoder to extract object features from the images, and use the object features along with the generated text to fine-tune the T2I model.

    • Inference Stage One-Shot, No need of ReFine-Tune

  • Deep Insight:

    • It's a single general model that is able to compose a new scene around given objects yet without the need of per-object optimization, using as few as one image

    • But it's very very Domain-Specific, and is really difficult to generalize to Open-Domain

DreamTuner

  • Main Method:

    (a) Subject Encoder Pre-training (b) Subject Encoder Attention

    • Subject Encoder: Deploy a trainable Subject-Encoder to extract Subject features from reference images. Insert a Subject-Encoder-Attention layer (like the original Cross-Attention, the residual blocks is used to adjust the feature dimension) between the SD's SA layer and CA layer to fuse Subject features into the generation process, thus achieving a coarse injection of new concepts. To decouple content and background features, SOD is used to remove the background; to decouple content and position features, a pre-trained ControlNet can be introduced to input positional information, ensuring that the learned features are content-specific.

    (c) Self Subject Attention

    • Self-Subject-Attention The Self-Subject-Attentionis formulated as follows

      (1)S-S-A(z,zr)=Softmax(q⋅[k,kr]Td+log⁡m)⋅[v,vr]w.r.t.kr=Key(zr),vr=Value(zr)m=[J,ωref∗mr]

      where zr is the reference image features, [ ., .] indicates concatenation, mr is the foreground mask of reference image, J is a all-ones matrix, ωref is an adjustment coefficient.

BLIP-Diffusion

  • Main Method:

    • Multimodal Representation: Introducing BLIP-2, a Multimodal-Encoder is pre-trained using a large-scale Image-Text dataset to obtain image features that are perfectly aligned with the text.

    • Subject Representation: Given a subject image, training images are constructed by replacing the background, and then input training images into the Multimodal-Encoder to obtain Subject Prompt Embeddings. Given the Subject Text corresponding to the Subject Image, the Text-Prompts are constructed in conjunction with the Subject Prompt Embeddings. The T2I model is then fine-tuned using the Subject Image and the constructed Text-Prompts.

Cones V2

  • Main Method:

    • For subjects belonging to a certain class, Cones2 employs a training loss from Textual Inversion to learn an offset for the Subject Embedding relative to the class embedding, termed the residual text embedding. Concurrently, regularization is ensured by maintaining the constancy of all other Token Embeddings in the Prompt, with the exception of the Subject Token Embedding.

    • Cross-Attention Control: EditedCA(CA,M,c)=Softmax(CA⊕(η(t)⋅Msi|i=1,…,N))⋅(Wv⋅E(c))
